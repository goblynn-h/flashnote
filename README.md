# 闪记

极简 Windows 闪记工具——按下全局快捷键，屏幕中央弹出输入框，打字、回车，内容即存入当日 Markdown 文件。

## 特性

- **一键呼出**：任意应用内按下快捷键，输入框居中弹出，自动获取焦点
- **即打即存**：输入内容回车保存，追加到按日期命名的 Markdown 文件
- **系统托盘**：常驻托盘，右键菜单可修改文件夹、快捷键、开机自启
- **快捷键捕获**：修改快捷键时直接按键盘，自动识别组合键，无需手写语法
- **纯本地**：不联网、不注册、不弹广告，数据完全在本地
- **轻量**：编译后单文件约 1.3MB，运行内存 <5MB，无需管理员权限

## 使用

| 操作 | 说明 |
|------|------|
| 呼出输入框 | 按下快捷键（默认 `Ctrl+Shift+J`） |
| 提交笔记 | `Enter`，内容写入 `笔记目录/YYYY-MM-DD.md` |
| 放弃输入 | `Esc` |

每条记录格式：

```
2026/6/2 14:30 朋友啊，你吃饱未？
```

文件按日期命名，同一天的笔记追加在同一文件内。

### 首次运行

双击 `FlashNote.exe`，依次弹出：

1. **选择存储文件夹**——所有笔记将保存在此目录
2. **按下快捷键**——直接敲键盘设置（默认 `Ctrl+Shift+J`，按 `Esc` 使用默认）

设置完毕自动启用开机自启，写入同目录 `config.ini`。

### 托盘菜单

右键系统托盘「闪记」图标：

- **修改存储文件夹**——弹出文件夹选择对话框
- **修改快捷键**——弹窗后直接按键盘，程序自动识别
- **开机自启 ✓**——点击切换，在启动文件夹创建/删除快捷方式
- **退出**——关闭程序

## 从源码编译

### 环境准备

安装 [AutoHotkey v2](https://www.autohotkey.com)（注意是 **v2**，不是 v1.1）。

### 编译步骤

1. 打开 Ahk2Exe（开始菜单搜索 "Convert .ahk to .exe"）
2. 填写以下路径：

| 字段 | 值 |
|------|-----|
| Source (脚本文件) | `<项目路径>/FlashNote.ahk` |
| Destination (输出文件) | `<项目路径>/FlashNote.exe` |
| Base file (.bin) | `C:\Program Files\AutoHotkey\v2\AutoHotkey64.exe` |
| Custom Icon (.ico) | （可选）选择一个 .ico 图标文件 |

3. 点击 **Convert**

编译完成后即可分发 `FlashNote.exe`，目标电脑无需安装 AHK。

### 配置文件

首次运行后自动生成 `config.ini`：

```ini
[Settings]
NoteDir=D:\MyNotes
Hotkey=^+j
```
